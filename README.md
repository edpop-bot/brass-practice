# 楽器練習アプリ

楽器（トランペット・チューバ）の音符練習用Webアプリです。

## 主な機能
- 楽器・レベル・オクターブ・半音の出題条件指定
- 音符画像と指番号・記譜音・実音の表示
- オートモード（自動出題）
- スマホ対応

## 使い方
1. `index.html` をブラウザで開くだけで利用できます。
2. 画像ファイルは `assets/` フォルダに配置してください。

## 公開方法
GitHub Pagesで公開する場合は、
- このリポジトリをGitHubにpush
- リポジトリの「Settings」→「Pages」で`main`ブランチの`/`を公開対象に設定
